# 書架空少年帰（Shosho）

バーチャル書店で、本との偶然の出会いを。

## About

Shoshoは、リアル書店での「探索の楽しさ」をオンラインで再現するバーチャル書店体験サービスです。

価格や在庫ではなく、**表紙との出会い**から本を選ぶ。そんな少年時代の本屋体験を、3D空間で蘇らせます。

## Features

- **バーチャル書店空間** - ノスタルジックな3D空間で本棚を巡る
- **ランダム落下場** - 偶然の出会いを演出する特別な空間
- **表紙買い体験** - 価格や在庫に惑わされない純粋な本選び
